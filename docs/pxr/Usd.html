SetLimits(limits: {subDictKey: {entryKey: Any}}) → bool#

Sets the limits metadata for this attribute.

Parameters: limits ({subDictKey: {entryKey: Any}}) - The limits dictionary to author for this attribute. subDictKey and entryKey are Python str keys. Common built-in sub-dictionary keys include Usd.LimitsKeys.Hard and Usd.LimitsKeys.Soft; common entry keys include Usd.LimitsKeys.Minimum and Usd.LimitsKeys.Maximum, but custom sub-dictionaries and custom entries are also allowed.

Returns: bool - True if the limits metadata is authored successfully; otherwise False.

Example:

from pxr import Usd, Sdf
stage = Usd.Stage.CreateInMemory()
prim = stage.DefinePrim("/MyPrim")
attr = prim.CreateAttribute("attr", Sdf.ValueTypeNames.Int)

result = attr.SetLimits({
    Usd.LimitsKeys.Soft: {
        Usd.LimitsKeys.Minimum: 5,
        Usd.LimitsKeys.Maximum: 10,
    },
    Usd.LimitsKeys.Hard: {
        Usd.LimitsKeys.Minimum: 0,
        Usd.LimitsKeys.Maximum: 15,
    },
    "myCustomSubDict": {
        Usd.LimitsKeys.Minimum: 50,
        Usd.LimitsKeys.Maximum: 100,
        "customKey": 42.5,
    },
})
